## TAM, SAM, SOM

- **TAM** (productivity + knowledge activation + personal capture/recall): **$60B–$150B**
- **SAM** (knowledge workers, prosumers, teams): **$20B–$50B+**
- **SOM** (3-year wedge: keyboard-first power users + early teams): **Tens to low hundreds of millions** in annual spend potential

Reaching $10M ARR requires only a modest foothold within SAM at pricing comparable to adjacent tools (~$8–$20/user/month).

## Competitive Landscape

### Exploitable Gap

**Event-based triggers tied to digital context**, expressed in the user's language, with minimal friction. Even large ecosystems show uneven support for contextual triggers; transitions can remove features. EasyJot competes by treating **context triggers as the core product**, not an add-on.

### Competitor Clusters

| Cluster | Example | EasyJot Wedge |
|---------|---------|---------------|
| Workspaces | Notion | Cross-workflow context trigger layer vs. pull-based workspace |
| Native reminders | Apple Reminders | Digital context triggers + semantic memory |
| Task managers | Todoist | Intent objects vs. dated tasks; app/site/contact triggers |
| Launchers | Raycast | Persistent intent recall that triggers itself |
| AI notes | Mem | Explicit workflow triggers vs. "related content" |
| Ambient capture | Rewind/Limitless | Intent-only, user-authored vs. broad surveillance |
| Suite copilots | Microsoft 365, Google | Cross-workflow, vendor-agnostic layer |

## Moat and Defensibility

1. **Personal intent data graph** — User-authored, high-signal; structurally difficult to replicate without being the daily capture surface.

2. **Behavioral learning loop** — Trigger feedback (act, snooze, dismiss, irrelevant) improves precision over time; "why this surfaced" explanation turns model into user-tunable system.

3. **Workflow context engine** — Engineering-heavy, permission-compliant, cross-platform. Accessibility trust on macOS and mobile constraints create an engineering moat.

4. **Switching costs / capture habit** — Keyboard-first capture becomes muscle memory; switching away loses "where intentions go."

5. **Integration depth** — Calendar, browser, comms integrations create value and switching costs.
