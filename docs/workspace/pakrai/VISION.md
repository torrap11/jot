# PakrAI — vision

**Scope:** [SCOPE.md](./SCOPE.md)

---

## One line

Write notes as you work; **query your screen** when memory fails; PakrAI **resurfaces** the right note by time or context—and **Pakr** helps you reorganize when you ask.

---

## What PakrAI is

A **local-first macOS app** centered on **notes**, with:

- **Screen recording** in the background (local, for your eyes only)
- **Query recordings** when you need “what was on my screen?”
- **Ambient note resurfacing** (calm, note-first)
- **Pakr** for note cleanup

Not a meeting recorder. Not camera. Not chat-first.

---

## Core behaviors

1. **Capture notes** while working  
2. **Search notes** (⌘P)  
3. **Query screen recordings** — search/ask over captured desktop text  
4. **Resurface notes** — time or work context  
5. **Pakr agent** — reorganize notes on demand

## Goals

| # | Goal |
|---|------|
| G1 | Trustworthy note resurfacing |
| G2 | Time + context resurfacing |
| G3 | Fast note search |
| G4 | **Reliable recording + useful query** |
| G5 | Pakr for reorg |
| G6 | Local-first, calm defaults |

---

## Non-goals (v1)

Microphone, camera, cloud sync, LLM on proactive hot path, separate Screenpipe app for daily use.

---

## Principles

1. **Notes are the hero** — recording supports recall, not the other way around  
2. **Query when you want** — resurfacing when it’s confident  
3. **Silence by default** on proactive path  
4. **Two speeds** — ambient vs Pakr / query panels
